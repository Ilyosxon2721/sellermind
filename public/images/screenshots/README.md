# PWA Screenshots

Скриншоты для улучшенного install prompt в PWA.

## Требуемые файлы

### 1. dashboard-wide.png
- **Размер:** 1280x720 px
- **Формат:** PNG
- **Описание:** Скриншот дашборда в широком формате (desktop/tablet)
- **Содержимое:** Главная панель управления SellerMind с графиками и виджетами

### 2. dashboard-narrow.png
- **Размер:** 750x1334 px
- **Формат:** PNG
- **Описание:** Скриншот дашборда в узком формате (mobile)
- **Содержимое:** Мобильная версия дашборда SellerMind

## Как создать скриншоты

1. Откройте приложение в браузере
2. Для wide: установите размер окна 1280x720
3. Для narrow: используйте DevTools для эмуляции iPhone (750x1334)
4. Сделайте скриншоты и сохраните в этой директории

## Использование

Скриншоты используются в PWA manifest для показа превью приложения при установке.
Браузеры показывают эти изображения в диалоге установки PWA.
