Realization Sales Report{{ /api/v5/supplier/reportDetailByPeriod }}

https://statistics-api.wildberries.ru/api/v5/supplier/reportDetailByPeriod
Описание метода

Details for the realization reports.

The report contains data since 29 January 2024.

You can save the report in Google Sheets
Request limit per one seller's account:
Period Limit Interval Burst
1 minute 1 request 1 minute 1 request
Authorizations:
HeaderApiKey
query Parameters
dateFrom
required
string <RFC3339>

Starting date of the report.
Date format: RFC3339. You may send date or date with time. Time could be specified in seconds or milliseconds.
The time stands in Moscow time zone (UTC+3).
Examples:

  • 2019-06-20
  • 2019-06-20T23:59:59
  • 2019-06-20T00:00:00.12345
  • 2017-03-25T00:00:00
dateTo
required
string <date>

Report end date

limit
integer <= 100000
Default: 100000

Number of strings in the response

rrdid
integer
Default: 0

The unique ID of the report line. Required to receive the report in parts.
Report loading must start with rrdid = 0 and on subsequent API calls, pass the rrd_id value in the request from the row line received as a result of the previous call.
Thus, to load a single report, you may need to call the API until the response returns an empty array [].

period
string
Default: "weekly"
Enum: "weekly" "daily"

Report periodicity:

  • weekly
  • daily

Documents List{{ /api/v1/documents/list }}

https://documents-api.wildberries.ru/api/v1/documents/list
Описание метода

Returns seller's documents list

Request limit per one seller's account:
Period Limit Interval Burst
10 seconds 1 request 10 seconds 5 requests
Authorizations:
HeaderApiKey
query Parameters
locale
string
Default: "en"
Example: locale=ru

category field language:

  • ru — Russian
  • en — English
  • zh — Chinese
beginTime
string <date>
Example: beginTime=2024-07-09

Period start. Only with endTime

endTime
string <date>
Example: endTime=2024-07-15

Period end. Only with beginTime

sort
string
Default: "date"
Enum: "date" "category"
Example: sort=category

Sorting:

  • date — by the document creation date
  • category — by category (only when locale=ru)

Only with order

order
string
Default: "desc"
Enum: "desc" "asc"
Example: order=asc

Data order:

  • desc — descending
  • asc — ascending

Only with sort

category
string
Example: category=redeem-notification

Document category ID from the name field

serviceName
string
Example: serviceName=redeem-notification-44841941

Unique document ID

limit
integer <= 50
Default: 50
Example: limit=10

The maximum number of response rows

offset
integer
Default: 0
Example: offset=90

From which row to start outputting data
