# SellerMind PWA - Техническое задание

> **Версия:** 1.0
> **Дата:** 2026-03-08
> **Цель:** Полная PWA адаптация всех страниц приложения с нативным дизайном

---

## Общие требования

### Технологии
- **CSS:** Модульная система `resources/css/pwa/` (variables, base, components, skeleton, pages)
- **JS:** Alpine.js 3.x с persist
- **Режимы:** `.pwa-mode` (standalone) и `.browser-mode` (web)
- **Платформы:** iOS (`.pwa-ios`) и Android (`.pwa-android`)

### Принципы дизайна
1. **Mobile-first** — все страницы оптимизированы под телефоны
2. **Native feel** — iOS/Android нативные паттерны
3. **Safe areas** — поддержка notch, home indicator
4. **Gestures** — pull-to-refresh, swipe actions
5. **Haptic** — вибрация на действиях
6. **Offline** — работа без сети где возможно

### Компоненты дизайн-системы
- `sm-header` — нативный header с back button
- `sm-tabbar` — нижняя навигация (5 табов)
- `sm-card` — карточки контента
- `sm-list` — списки с chevron
- `sm-btn` — кнопки (primary, secondary, danger)
- `sm-search` — поиск с иконкой
- `sm-filter-row` — горизонтальные фильтры
- `sm-bottom-sheet` — модальный sheet снизу
- `sm-fab` — floating action button
- `sm-badge` — статусные бейджи

- [ ] **Gestures** — swipe actions где нужно
- [ ] **Forms** — нативные input types

- [ ] **Images** — lazy loading
